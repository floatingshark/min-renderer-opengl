# Mock Renderer OpenGL
[![License: MIT](https://img.shields.io/badge/License-MIT-yellow.svg)](https://opensource.org/licenses/MIT)  
A simple rendering application using C++ and OpenGL.  

# Requirement
Compiler MinGW-W64-builds-5.0.0  
OpenCV

# Usage
```
$cd {project root}  
$make
```
